Official Wix Developer Resources
The best place to start is always the official documentation and learning platforms. Wix has a dedicated ecosystem for app developers.
 * Wix Developers Documentation - Build Wix Apps: This is your central hub for all things related to building Wix apps. It provides a clear overview of the different frameworks you can use to build apps, including the Wix Blocks and the Wix CLI (Command Line Interface). It also provides quick-start tutorials and documentation for the SDK itself.
   * URL: https://dev.wix.com/docs/build-apps
 * Wix Developers Tutorial: Set Up an App With the App Dashboard: This is a crucial, step-by-step tutorial that walks you through the process of setting up a self-hosted app using the App Dashboard in your Wix Studio workspace. It shows you how to create the app, add extensions, and install it on a development site.
   * URL: https://dev.wix.com/docs/build-apps/get-started/tutorials/tutorial-set-up-an-app-with-the-app-dashboard
 * Wix Developers Tutorial: Create a Self-Hosted App: This guide provides the foundational steps for building a self-hosted app, which is a common approach for custom app development. It explains how to create the app in your Custom Apps page and add a dashboard page extension.
   * URL: https://dev.wix.com/docs/build-apps/get-started/quick-start/create-a-self-hosted-app
 * Wix Studio: Creating Custom Apps: This Help Center article provides a high-level overview of the process, from creating a custom app in your Wix Studio workspace to publishing it on the Wix App Market. It also covers the different building frameworks available.
   * URL: https://support.wix.com/en/article/wix-studio-creating-custom-apps
 * Wix SDK Documentation: While not a course, this is an essential reference. It details the various modules and APIs within the Wix JavaScript SDK that you'll use to build your app's functionality, including backend and frontend modules for interacting with business solutions, data, and more.
   * URL: https://dev.wix.com/docs/sdk
