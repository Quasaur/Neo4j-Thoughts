For the **Stripe Test Card** (

```
4242 4242 4242 4242
```

), you can use:

- **Expiration Date:** Any valid future date (e.g., 
    
    ```
    12/30
    ```
    
     or 
    
    ```
    01/28
    ```
    
    )
- **CVC:** Any 3 digits (e.g., 
    
    ```
    123
    ```
    
    )
- **ZIP Code:** Any valid ZIP (e.g., 
    
    ```
    10001
    ```
    
    )
