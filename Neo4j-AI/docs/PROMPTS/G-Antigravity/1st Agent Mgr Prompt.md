All three workspaces (thoughts, book-of-thoughts, wisdom-book) are versions of the same project.

The workspace labeled "thoughts" is the original version, published to (https://quasaur.github.io/thoughts/). It uses Typescript and a static site generator called Quartz (https://github.com/jackyzha0/quartz).

The workspace labeled "book-of-thoughts" is the second version, developed using Django/Python, the Neo4j AuraDB platform, Typescript and ReactJS.

The workspace labeled "wisdom-book" is the third and latest version, developed using Django/Python, the Neo4j AuraDB platform, Typescript, ReactJS and Tailwind CSS. "wisdom-book" is the primary focus of current development efforts.

You will analyze all three projects.
